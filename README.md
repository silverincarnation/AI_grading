This is a simple optimization example designed to demonstrate the basic workflow of the grading system. The project includes a problem statement, a student solution, and a reference correct answer.

If needed, additional LangGraph features can be incorporated in the future, such as time travel or router.

Initially, the plan was to use SymPy to verify correctness by checking whether the left-hand side and right-hand side of equations are equal. However, this approach does not always hold.
